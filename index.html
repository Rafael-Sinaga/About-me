Tentang Saya

Saya adalah mahasiswa aktif Program Studi Computer Science - Software Engineering dengan minat kuat pada pengembangan web dan desain antarmuka. Saya memiliki keahlian dalam HTML, CSS, JavaScript, serta Figma untuk desain UI/UX. Selain itu, saya juga berpengalaman dalam automation testing menggunakan Selenium WebDriver.

Sejak pertengahan 2024 hingga saat ini, saya aktif sebagai freelancer dalam bidang desain web, membantu berbagai klien dalam mewujudkan tampilan website yang menarik dan user-friendly. Passion saya dalam teknologi dan desain terus mendorong saya untuk terus belajar dan berkembang menjadi seorang software engineer yang berkualitas dan mampu berkontribusi positif dalam dunia digital.
